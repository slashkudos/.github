## How It Works

### Giving Kudos

slashkudos integrates into the apps you already know and love, so no need to learn something new. In Discord, GitHub, Teams, and Slack where the slashkudos app is installed, just type `/kudos Thanks @philip-gai for being awesome! ðŸš€` to give kudos.

Twitter works a bit differently - just Tweet [@slashkudos][slashkudos Twitter]. For example, tweet `@slashkudos Thanks @philip-gai for being awesome! ðŸš€`.

### Exploring Kudos

Visit [app.slashkudos.com] where you can:

- View kudos you have received
- View kudos you have given
- View a universal feed of kudos
